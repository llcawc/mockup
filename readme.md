# Mockup Starter template

_Version 0.0.1 date 24.01.21_

> Bootstrap starter template for local development a multi-page site, witch use Gulp, Dart SASS, PostCSS, Rollup and Imagemin.

FrontEnd of html based on layouts and partials of [Handlebars (panini compiler)](https://github.com/foundation/panini) templates.

Used technologies: HTML5, SASS/SCSS, Module JS, Bootstrap. Bootstrap-Icons. Lightbox. Social share button code include.

You can works both in development mode use css-maps and light local static server and in production mode who used purges, prefixs and minification.
